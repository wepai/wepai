NPWPanda Landing Page Bundle
=============================

Berkas dalam paket ini:
- index.html
- favicon.png
- favicon.ico
- logo.png
- og-image.jpg
- qris.jpeg (placeholder — ganti dengan QRIS asli Anda)

Petunjuk singkat:
1. Upload SEMUA berkas ke root folder hosting (mis. public_html) sehingga URL menjadi:
   - https://npwpanda.info/index.html
   - https://npwpanda.info/favicon.png
   - https://npwpanda.info/favicon.ico
   - https://npwpanda.info/logo.png
   - https://npwpanda.info/og-image.jpg
   - https://npwpanda.info/qris.jpeg

2. Di index.html, meta og:image mengarah ke https://npwpanda.info/og-image.jpg — biarkan begitu agar sharing di sosial media terbaca benar.

3. Ganti qris.jpeg dengan gambar QRIS Anda (nama file sama).
